Understanding HTML & Semantic HTML

What is HTML? What is Semantic HTML? Common Semantic Tags

What is HTML?

HTML is the standard markup language for creating web pages. It defines the structure and content of a webpage using a set of tags and attributes. Browsers interpret the HTML code and render the content accordingly, displaying text, images, videos, and other elements on the webpage.

What is Semantic HTML?

Semantic HTML goes beyond just defining the structure of a webpage. It uses specific tags that describe the meaning and purpose of the content they contain. This makes the content more understandable for users, search engines, and assistive technologies.

Commonly Used Semantic Tags

  • <header>: Defines the header section of a webpage (you're using it now!)
  • <nav>: Defines a section of navigation links
  • <main>: Defines the main content area of a webpage
  • <article>: Defines a self-contained piece of content, like a blog post
  • <section>: Defines a thematic grouping of content
  • <aside>: Defines content that is indirectly related to the main content
  • <footer>: Defines the footer section of a webpage (you're using it now!)
